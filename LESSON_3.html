Repetition While and For

Repetition Structure

โครงสร้างแบบวนซ้ำ, โครงสร้างการทำงานที่มีการทำงานในบางคำสั่งหรือบางชุดของคำสั่งซ้ำกันมากกว่าหนึ่งรอบขึ้นไป โดยการวนซ้ำนี้จะต้องมีการตัดสินใจร่วมอยู่ด้วยเสมอ เพื่อเป็นเงื่อนไขที่จะตัดสินใจว่าเมื่อใดจะวนซ้ำ หรือเมื่อใดจะถึงเวลาหยุดการวนซ้ำ

While Loop

While เป็นคำสั่งวนซ้ำที่มีการตรวจสอบเงื่อนไข (condition) ก่อนเข้าทำงานเสมอ เมื่อเงื่อนไขที่ ทำการตรวจสอบเป็นจริง จึงจะประมวลผลคำสั่งหลัง while แต่ถ้าเงื่อนไขเป็นเท็จจะยุติการทำงานทันที

Example (While Loop)

While condition
statement(s)
statement(s)
...
statement(s)

For Loop

คำสั่ง for เป็นคำสั่งที่ใช้สำหรับการทำซ้ำ เช่นเดียวกับ while และต้องมีการตรวจสอบเงื่อนไข ก่อนเข้าลูปเหมือนกัน แต่แตกต่างกันตรงที่ for จะตรวจสอบรายการแบบลำดับแทน (Item of sequence) เช่น ข้อมูลชนิดสตริง ลิสต์ หรือทัพเพิล

Example (For Loop)

for iterating var in sequence:
statement(s)

Calculating Running Total

ยอดรวมคือผลรวมของตัวเลขที่สะสมจากการวนซ้ำแต่ละครั้ง ตัวแปรที่ใช้เก็บยอดรวมเรียกว่าตัวสะสม

โปรแกรมที่คำนวณผลรวมของชุดตัวเลขมักใช้สององค์ประกอบ:

- วงวนที่อ่านแต่ละตัวเลขในชุด

- ตัวแปรที่สะสมตัวเลขทั้งหมดเมื่ออ่าน

Sentinel

Sentinel เป็นค่าพิเศษที่ทำเครื่องหมายจุดสิ้นสุดของลำดับของค่า

- เพียงถามผู้ใช้เมื่อสิ้นสุดการวนซ้ำแต่ละครั้ง หากมีค่าอื่นที่ต้องดำเนินการ อย่างไรก็ตาม หากลำดับของค่ายาว อย่างไรก็ตาม การถามคำถามนี้เมื่อสิ้นสุดการวนซ้ำแต่ละครั้งอาจทำให้โปรแกรมยุ่งยากสำหรับผู้ใช้

- ถามผู้ใช้ที่จุดเริ่มต้นของโปรแกรมว่ามีกี่รายการในลำดับ อย่างไรก็ตาม สิ่งนี้อาจทำให้ผู้ใช้ไม่สะดวก หากลำดับนั้นยาวมากและผู้ใช้ไม่ทราบจำนวนรายการที่อยู่ในนั้น ผู้ใช้จะต้องนับจำนวนรายการนั้น

Example (Sentinel)

loop = 'y'
while loop =='y'
num = int(input('Enter a number:'))
if num %2 == 0:
print('This number is even number')
else :
print('This number is odd number')
loop = input('Again? (Enter y for yes):')

This number is odd number
Again? (Enter y for yes):y
Enter a number 2
This number is even number
Again? (Enter y for yes):y

Input Validation Loops

การตรวจสอบความถูกต้องของข้อมูลเข้าเป็นกระบวนการตรวจสอบข้อมูลที่ป้อนเข้าสู่โปรแกรม เพื่อให้แน่ใจว่าถูกต้องก่อนที่จะนำไปใช้ในการคำนวณ การตรวจสอบความถูกต้องของอินพุตมักทำโดยใช้ลูปที่วนซ้ำตราบเท่าที่ตัวแปรอินพุตอ้างอิงข้อมูลที่ไม่ถูกต้อง

Example (Input Validation Loops)

score = int(input('Enter a test score: '))
while score < 0 or score > 100:
print('Error: The score cannot be
negative or greater than 100.')
score = int(input('Enter a test score: '))

Resultant (Input Validation Loops)

Enter a test score: -1
Error: The score cannot be negative or greater than 100.
Enter a test score: 101
Error: The score cannot be negative or greater than 100.
Enter a test score: 100

Nested loops

ไพธอนให้ผู้เขียนโปรแกรมสามารถใช้ลูปซ้อนได้ (การวาง for loop ไว้ภายใน for loop) มีรูปแบบคำสั่งดังนี้

ลูปซ้อนของ for

for iterating_var insequence:
for iterating_var insequence:
statements(s)
statements(s)

ลูปซ้อนของ while

while expression:
while expression:
statement(s)
statements(s)

ลูปซ้อนของ for กับ while

for iterating_var insequence:
while expression:
statement(s)
statements(s)

การวางลูปซ้อนภายใน ไพธอนไม่ได้จำกัดว่ามีได้ทั้งหมดกี่ชั้น ดังนั้นผู้เขียนโปรแกรมสามารถ วางลูปซ้อนได้ตามอิสระ ขึ้นอยู่กับปัญหาว่ามีความซับซ้อนมากน้อยเพียงใด แต่สิ่งที่ควรจดจำไว้เสมอ คือ การวางลูปที่ซ้อนกันมากๆ จะทำให้โปรแกรมอ่านหรือแก้ไขข้อผิดพลาดได้ยากขึ้นด้วย จะเกิดปัญหา เรื่องของการย่อหน้าภายในลูป อาจจะส่งผลให้โปรแกรมทำงานผิดพลาด ตรงจุดนี้ถือว่าเป็นจุดอ่อน ของไพธอนก็ได้ เพราะการไม่มีเครื่องหมายบอกขอบเขต ทำให้บางครั้งต้องใช้สายตาในการหาขอบเขตของคำสั่งเอง

Example (Nested loops)

for i in range(1, 10):
print("For loop for i")
for j in range(1, i):
print("For loop for j")

Continue

Break

เป็นคำสั่งที่สั่งให้โปรแกรมยุติการทำซ้ำ(Loop) สำหรับการทำงานของคำสั่ง break ใน nested loop เมื่อโปรแกรมยุติการทำงานใน Loop ปัจจุบันแล้ว โปรแกรมจะทำงานต่อใน Loop ที่ครอบ Loop ปัจจุบันอยู่ต่อไป คำสั่ง break จะใช้ทำงานกับ while และ for loop เท่านั้น ไม่สามารถใช้ได้กับ if, if-else, if-elif ได้เหมือนในภาษาซีสำหรับแผนผังจำลองการทำงานของ break
