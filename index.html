What is Python?
Python is an advanced computer programming language. It is designed to be a simple scripted language b y eliminating the complexities of the language's structure and grammar. Python provides a function called "Interpreter". It translates a set of instructions that we write into machine language line by line to input into the Computer system in order for the computer to function as we desire.

Why Python?
Python is more widely used to develop websites than other languages because it is simple to understand.
